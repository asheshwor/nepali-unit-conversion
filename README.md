# Nepali unit conversion

A tool for converting between diff units of length and area common to Nepal built using JavaScript. This includes conversion between Nepali units like Ropani, Paisa, Kattha etc. The factors are based on the Statistical Pocket Book published by the [Central Bereau Of Statistics] (http://cbs.gov.np/publications/Statistical%20Pocket%20Book%202014), Kathmandu, Nepal.

The JavaScript code was written in the early 2000s but still seem to work on modern browsers.
